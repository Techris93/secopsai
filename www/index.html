Capabilities

Detect, correlate, and triage — locally

SecOpsAI is a local-first security monitoring and correlation platform. It combines OpenClaw telemetry with host OS events to surface higher-signal findings without shipping your logs by default.

Universal adapters

Collect from OpenClaw, macOS, Linux, and Windows using a shared adapter model and normalize events into one schema.

Cross-platform correlation

Correlate findings by IP, user, time window, and file hash to identify multi-system attack patterns.

Operator workflows

Use the CLI, OpenClaw plugin, and WhatsApp workflows to review, triage, and respond quickly.

Local-first storage

Findings persist to a local SQLite SOC store so you can keep data on your own machine by default.

Quick Start

Zero to findings in minutes

curl -fsSL https://secopsai.dev/install.sh | bash
cd ~/secopsai && source .venv/bin/activate
secopsai refresh
python3 cli.py refresh --platform macos,openclaw
python3 cli.py correlate
secopsai list --severity high
