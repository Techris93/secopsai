Quick Start

Install SecOpsAI

Works everywhere. Installs everything.

One-liner install for macOS

curl -fsSL https://secopsai.dev/install.sh | bash

The one-liner installs Python, dependencies, and everything else for you.

Install via npm

For Node.js users who prefer package managers.

npm install -g secopsai

Requires: Node.js 18+

Windows via WSL2

Required

SecOpsAI requires a Linux environment. Windows users must use WSL2 (Windows Subsystem for Linux).

Step 1: Install WSL2 + Ubuntu
wsl --install -d Ubuntu
Step 2: Install SecOpsAI in WSL
curl -fsSL https://secopsai.dev/install.sh | bash

Requires: Windows 10/11 · WSL2 · Ubuntu 22.04+
