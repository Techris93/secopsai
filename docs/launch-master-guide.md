# Launch Master Guide

This page is the short-form operator view of the full launch package.

## Core Deliverables

- Product documentation for installation, rules, API, and deployment
- Container packaging and Render deployment path
- GitHub Actions workflows for build, benchmark, and security scanning
- Marketing and launch planning documents in the repository root

## Suggested Execution Order

1. Validate code and benchmark locally
2. Confirm CI status on `main`
3. Confirm worker deployment and findings persistence
4. Confirm docs deployment and custom domain routing
5. Execute public launch checklist
